Development project

Dit project heb ik samen uitgevoerd met Jarvin. Jarvin heeft meer ervaring met front-end development dan ik. Aan het einde van de brandingfase hadden we al afgesproken om samen een groepje te vormen, mogelijk samen met Noortje. We waren enthousiast over het idee om met een framework te gaan werken. Jarvin had hier al veel ervaring mee, terwijl dit voor mij grotendeels nieuw terrein was.

We besloten om gebruik te maken van het framework Next.js. Ook spraken we af om het design van Jarvins vorige projectgroep te gebruiken en dit gezamenlijk te coderen.

Taakverdeling

Ik was verantwoordelijk voor het coderen van de welkomspagina, footer, navigatie tussen stations, visuele pagina en auditieve pagina. De navigatie tussen stations bleek uiteindelijk te complex voor mij, waardoor Jarvin deze taak heeft overgenomen.

Een groot deel van deze code heb ik opgezet met hulp van ChatGPT, maar ik begreep steeds beter hoe de code werkte en waarom bepaalde keuzes werden gemaakt. Vooral het schrijven van functies vond ik in het begin lastig, evenals het werken met TypeScript, waar ik nog weinig ervaring mee had.

et schrijven van HTML ging mij wel goed af, dat kon ik zelfstandig doen. Bovendien kreeg ik waardevolle uitleg en ondersteuning van Jarvin. Wanneer ik ergens tegenaan liep, kon ik altijd bij hem terecht. Zo heeft Jarvin onder andere de aanpak bedacht voor de vertaalfunctionaliteit binnen de site. Dankzij zijn uitleg begreep ik het direct en kon ik het zelfstandig toepassen in de code.

Github

Voor het samenwerken hebben Jarvin en ik gebruik gemaakt van Github. Hierbij hebben we gewerkt met versies en branches en versies. Jarvin gaf me ook nog een tip om vaker mijn veranderingen te committen. Hierbij kan je makkelijk naar andere versies mocht je iets makkelijk terug willen veranderen en kun je makkelijk zien in welk document er aanpassingen zijn gedaan. Hierbij staat niet elk document op modified.

Portfolio website

Binnen deze leeruitkomst heb ik mijn vaardigheden toegepast door het ontwikkelen van mijn portfolio-website met HTML, CSS en JavaScript. De website fungeert als een centrale plek waar ik mijn projecten en persoonlijke ontwikkeling presenteer. De site heeft inmiddels een duidelijke structuur en bevat al veel inhoud, maar is nog in ontwikkeling. Met name de styling en responsiveness kunnen nog worden verbeterd. Tijdens het ontwikkelproces heb ik gebruikgemaakt van Git en GitHub voor versiebeheer, zodat ik mijn code professioneel kon bijhouden en documenteren. Ik vond het werken met versies erg handig: zo kan ik eenvoudig teruggaan naar een eerdere versie als dat nodig is. Daarnaast zorgt het gebruik van versies ervoor dat Visual Studio Code alleen de relevante wijzigingen toont, wat bijdraagt aan overzichtelijkheid tijdens het ontwikkelen.

Gebruik van AI

Voor mijn portfolio heb ik een aantal keer gebruik gemaakt van AI met name ChatGPT.
Namelijk de volgende dingen:

  • Een navigatiebalk maken die bovenop andere elementen ligt (overlapping met content)
  • De delen van mijn naam bovenaan de pagina weergeven in een ander lettertype
  • Een carrousel/slider maken met meerdere afbeeldingen
  • De juiste HTML-structuur en beschrijving (alt-tekst) gebruiken bij afbeeldingen
  • Navigatie tussen verschillende leeruitkomsten op één pagina met JavaScript, zonder dat er aparte HTML-pagina’s nodig zijn.
