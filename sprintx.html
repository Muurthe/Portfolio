Project overview:

Voor mijn persoonlijke project wil ik een 3D-model maken van mijn kamer en deze op een website zetten. In dit model wil ik interactieve elementen en animaties toevoegen, zodat gebruikers kunnen navigeren en bepaalde objecten kunnen laten bewegen. Ik wil dit gebruiken als een soort portfolio van mezelf.


Voor dit project heb ik al een bootcamp gevold over het Framework Three.js en heb ik een tutorial van het leren van Blender gevolgd. Uitgewerkt verslag hiervan volgt.
